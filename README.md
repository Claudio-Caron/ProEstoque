# ProEstoque – Gestão Inteligente de Inventário

O ProEstoque é um sistema completo de gestão de produtos e controle de estoque desenvolvido para atender às necessidades de pequenos comércios. O projeto abrange desde a interface do usuário (Mobile/Web) até o desenvolvimento de uma API robusta com integração a banco de dados.

## Sobre o Projeto

Este software foi desenvolvido como projeto principal do semestre, focado em criar uma solução real e funcional para o mercado de varejo. Ele permite o controle total de movimentações, alertas inteligentes e visualização de dados através de dashboards dinâmicos.

## Funcionalidades Principais

* **Autenticação:** Fluxo completo de Login, Cadastro e Recuperação de Senha.
* **Gestão de Produtos:** Cadastro detalhado com suporte a fotos e categorização personalizada.
* **Controle de Estoque:** Histórico detalhado de movimentações (entradas e saídas).
* **Inteligência e Dados:** Dashboard com indicadores gráficos e notificações via push para níveis críticos de estoque.
* **Perfil do Usuário:** Gerenciamento de configurações e dados da conta.

## Tecnologias e Roadmap de Desenvolvimento

O projeto foi estruturado para evoluir de um protótipo funcional para uma aplicação Full Stack escalável, utilizando as seguintes tecnologias:

* **Frontend e Mobile:** Desenvolvimento de interfaces de alta fidelidade e setup moderno.
* **Backend:** Node.js com arquitetura API REST.
* **ORM:** Prisma para modelagem de dados.
* **Banco de Dados:** PostgreSQL.
* **Infraestrutura:** Planejamento de deploy e estratégias de integração contínua.
